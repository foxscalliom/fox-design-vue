# fox-design-vue

## 安装
```
npm install fox-design-vue
```

### 测试
```
yarn serve
```

### 构建测试文档
```
yarn run build
```

### 编译组件库
```
yarn lib
```
## 查询镜像
查询当前镜像地址：
	npm get registry
设置淘宝镜像：
	npm config set registry http://registry.npm.taobao.org/
设置官方npm镜像:
	npm config set registry https://registry.npmjs.org/
